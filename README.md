# gooplate

Use Google Sheets and Node.js to process templates

## Getting Started
Install the module with: `npm install gooplate`

```javascript
var gooplate = require('gooplate');

var goo = new Gooplate();
    
goo.generateOutputFromID( config.spreadsheet, config.sheet,
  function(err,pass)
  {
     
  }
);
```
